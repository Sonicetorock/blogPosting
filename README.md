# Project : blogPosting
## Tech Scack Used : 
- ExpressJS 
- MySQL
- Sequelize ORM
- Postman (For validity purpose)
- JWT authentication
## Before firing up the server : 
- run this command
  ``` npm i ```
- create a .env file
- add these below statements
 ```
PORT = 3000
JWT_SECRET = custom_secret(string)
DB_HOST = localhost
DB_USER = root
DB_PASSWORD = Your_SQL_password
DB_NAME = customDB_name
```
